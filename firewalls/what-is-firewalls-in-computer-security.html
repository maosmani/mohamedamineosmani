In computer security there are so may tools and software for protecting data, computers, networks, phones, and firewalls are one from them.

what is firewalls in computer?

In network computer security a firewall is device, or software that monitors the Incoming and outgoing network traffic.

Its main rule is to allow non-threatening traffic in and to keep dangerous traffic out the network.

A firewall sits as a gateway between a private internal network and the public Internet.

what does firewalls do?

A firewall is a security system that block the dangerous traffic from coming to the network, or to your computer, and allow the non-dangerous traffic to come inside the network, or to your computer.

It can detect, and block malware, and other types of dangerous security risks.
